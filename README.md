# Robot Kinematics Visualizer

[![Status](https://img.shields.io/badge/status-coming%20soon-yellow)](https://ecdaerol.com/resources/robot-kinematics)

Interactive serial manipulator kinematics — define DH parameters, visualize FK/IK, explore singularities.

## Planned features

- Define robot via Denavit-Hartenberg parameters
- Interactive 3D robot visualization
- Forward kinematics: set joint angles → end-effector pose
- Inverse kinematics: set target → solve joint angles
- Jacobian matrix display
- Singularity detection and manipulability ellipsoid
- Preset robots (2R, SCARA, 6-DOF industrial)
- Dark embed mode for portfolio integration

## Background

Based on ETH Zurich Robot Dynamics course (grade 6.0/6.0) with MATLAB and C# implementations.

## Tech stack

React · TypeScript · Vite · Tailwind CSS · Three.js
